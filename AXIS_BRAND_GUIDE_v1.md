# AXIS Brand Guide v1

AXIS should feel like a premium intelligent control system for business: calm, strong, structured, and technological without cheap futurism.

## Brand Character

AXIS is:
- intelligent system
- control center
- premium technology environment
- modern coordination layer
- order without dryness
- strength without aggression

AXIS is not:
- a household AI helper
- a cheap futuristic bot
- a boring CRM
- an acidic startup interface
- an overloaded AI illustration brand

## Colors

- AXIS Graphite: `#1B2430`
- AXIS Steel: `#344150`
- AXIS Silver: `#8C96A3`
- AXIS Mist: `#F3F4F6`
- AXIS White: `#FFFFFF`
- AXIS Blue Core: `#2F6BFF`
- Border Gray: `#D7DCE2`
- Soft Gray: `#B7C0CA`

Usage rule:
- 70% light and neutral tones
- 20% dark system tones
- 10% blue accent

## Typography

- Interface and body: Inter
- Brand headings: Sora or Space Grotesk

Headings should be confident and calm, with enough space. Body text should be readable and practical.

## UI Style

AXIS UI should be:
- clean
- premium
- structured
- calm
- easy to scan
- card-based, but not noisy

Buttons:
- Primary: Graphite or Blue Core background, white text
- Secondary: white background, graphite text, thin border

Icons:
- outline or semi-outline
- geometric
- not cartoonish

## Product Rule

The Mini App should look like a business command center, not like a technical Replit prototype.
